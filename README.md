# seithar-intel

**Threat Intelligence Skill — Seithar Group**

OpenClaw skill for automated collection, scoring, and delivery of cognitive threat intelligence.

## Capabilities

- RSS feed aggregation across 14+ threat domains
- Automated relevance scoring (cognitive warfare, cybersecurity, influence operations)
- DISARM framework mapping
- Channel delivery with deduplication
- Batch processing and temporal analysis

## Threat Domains

- **Cognitive Warfare** — State-level influence operations, narrative analysis
- **Cybersecurity** — Technical threats with cognitive dimensions
- **AI Safety** — AI-mediated manipulation, training data poisoning
- **OSINT** — Open source intelligence methodology

## Integration

Maps to Seithar Cognitive Threat Taxonomy (SCT-001 through SCT-012).
