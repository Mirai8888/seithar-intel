# seithar-intel

**Threat Intelligence Skill for OpenClaw**

Automated collection, scoring, and delivery of cognitive threat intelligence. Part of the Seithar Group defense infrastructure.

## Capabilities

- RSS feed aggregation across 14+ threat domains
- Automated relevance scoring (cognitive warfare, cybersecurity, influence operations)
- DISARM framework mapping
- Discord/channel delivery with deduplication
- Batch processing and temporal analysis

## Threat Domains

| Domain | Sources | Focus |
|--------|---------|-------|
| Cognitive Warfare | Atlantic Council, Frontiers in Psychology | State-level influence operations, narrative analysis |
| Cybersecurity | Krebs, BleepingComputer, SecurityWeek | Technical threats with cognitive dimensions |
| AI Safety | Various | AI-mediated manipulation, training data poisoning |
| OSINT | Bellingcat, Small Wars Journal | Open source intelligence methodology |

## Integration

This skill integrates with the Seithar Cognitive Defense Taxonomy (SCT-001 through SCT-012) for classification of detected threats.
